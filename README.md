## 不要传播，小心被拷打

### 环境安装

```bash
pip install -r requirements.txt
```



### config.json 配置

```json
{
    "user": "******",
    "pwd": "******",
    "login_url": "http://ydfwpt.cug.edu.cn/login/pre.html?continueurl=http://ydfwpt.cug.edu.cn/product/show.html?id=1",
    "target_url": "http://ydfwpt.cug.edu.cn/index.html"
}

```

在user和pwd处配置自己的账户和密码

其余的不用动
